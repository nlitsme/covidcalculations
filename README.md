COVID-19 stats
==============

Several html, javascript pages for displaying and calculating covid statistics:
 - Calculate the weekly incidence rate and estimate an effective R value for:
   - all [municipalities](https://rawcdn.githack.com/nlitsme/covidcalculations/71943384ffd2c4c43c0a25d01f4b574f018afbb4/rivm.html) in the Netherlands
   - all [countries](https://rawcdn.githack.com/nlitsme/covidcalculations/71943384ffd2c4c43c0a25d01f4b574f018afbb4/owid.html) in the World.
 - Estimate the date the various country's [vaccination](https://rawcdn.githack.com/nlitsme/covidcalculations/71943384ffd2c4c43c0a25d01f4b574f018afbb4/vac.html) programs will be done given
   the most recent progress.
